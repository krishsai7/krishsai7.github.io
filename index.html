Vehicular Accident Data from Montgomery County Police

Krishna Sai Kollipara

Overview

Vehicular accidents happen all over the world on a daily basis and learning to travel smart is always critical. For many people, travelling by car is the preferred means of transportion but the question of how safe is it always lingers. This tutorial will cover an analysis of traffic accident data collected by Montgomery County Police in the state of Maryland.

The reason for covering this data is in part to looking at a current goal by the department called Vision Zero. Vision Zero is a multinational road safety project that envisions a world where we no longer have traffic accidents or deaths. It originally started in Sweden in 1997 and has since been adopted by many countries around the world. Montgomery County adopted this goal and has the overarching goal to eliminate fatalities and severe injuries by 2030. Their short term goal for this project is to reduce collisions by 35% by November 2019.

Linear Regression Analysis

Unfortunately we only have 4 years of data to use for a linear regression. This is far from ideal and I would love to have more data before putting a lot of weight on this. Regardless, let's see what comes of it.

Linear Regression

I used the scikit-learn library to perform this linear regression. If you want to learn more, you can view an example about it at the following link: https://bigdata-madesimple.com/how-to-run-linear-regression-in-python-scikit-learn/

Full documentation can be found here: https://scikit-learn.org/stable/modules/generated/sklearn.linear_model.LinearRegression.html

reg = linear_model.LinearRegression()
X = [[x] for x in countsDF['Year'].values]
Y = [[y] for y in countsDF['Accident Count'].values]
regFit = reg.fit(X, Y)

predictedCrashes = []
for x in countsDF['Year'].values:
    predictedCrashes.append(regFit.predict(x)[0][0])

countsDF['Predicted Crashes'] = pd.Series(predictedCrashes, index = countsDF.index)

Conclusion

As mentioned previously, any result from this should be taken with a large grain of salt. I am only able to pass in 4 data points while 8 is the acceptable minimum. While the line is certainly trending downward, it is not necessarilly a valid test since I simply do not have enough yearly data to conduct this.

Based on the current data, we can see that accident counts are dropping but the question of if we will hit our goal outlined in Vision Zero by 2030 is yet to be seen. Until then, we need to focus engineering the roads, educating people, and forcing the law.
